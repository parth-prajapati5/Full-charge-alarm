# Full-charge-alarm
Full Charge Alarm is a simple yet useful application that alerts users when their device battery reaches 100% charge. This helps prevent overcharging, improves battery health, and ensures efficient power usage.

🚀 Features
🔔 Real-time battery monitoring
⚡ Instant alert when battery reaches full charge (100%)
🔊 Customizable alarm sound (optional)
📊 Lightweight and minimal resource usage
📴 Works in background
💡 Motivation

Overcharging devices can reduce battery lifespan over time. This project aims to provide a smart notification system that helps users unplug their device once it is fully charged.
